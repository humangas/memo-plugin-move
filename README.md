# memo-plugin-move
move is [mattn/memo](https://github.com/mattn/memo) plugin.  
Move the selected file to the specified directory

## Installation
```
$ go get -u -d github.com/humangas/memo-plugin-move
$ cd $GOPATH/src/github.com/humangas/memo-plugin-move
$ make install
```

## Usage
```
$ memo move <directory>
```
